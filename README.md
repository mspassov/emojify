# Emojify - AI Powered Emoji Converter
Convert any text to emojis, and then have the option to convert it back into text, based on the LLM's interpretation. 

## Key Learnings
- Next JS and React
- AI API integrations (Groq with Open AI model)
- RESTful APIs
- Drag and Drop library
- Server Actions, server / client components
- Vanilla CSS
